# DataForce Shiny App

This folder contains a starter Shiny app for browsing the six hackathon datasets.

## Run locally

From an R environment with required packages installed:

```r
setwd("shiny_app")
shiny::runApp()
```

## Required packages

- shiny
- readr
- dplyr
- ggplot2
- DT
- plotly
- tidyr
