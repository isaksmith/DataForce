## Application overview

DataForce is a Streamlit application built for the **Hack The Plains 2026** hackathon. It analyzes banking datasets to surface digital friction — the cost and frequency of failed or abandoned customer sessions — and provides tools for exploration, scoring, and simulation.

### Data pipeline

```
Hack The Plains 2026 Datasets/
├── branches.csv
├── customers.csv
├── digital_sessions.csv
├── error_codes.csv
├── feature_costs.csv
├── support_interactions.csv
└── session_friction_scores.csv   ← generated by friction_score_pipeline.py

artifacts/
├── customer_personas.json         ← generated by mirofish_export_pipeline.py
├── session_friction_scores.csv
└── simulation_seed_report.md
```

### Generating artifacts

```bash
# Generate friction scores (required for Friction Score page)
python3 friction_score_pipeline.py --output 'Hack The Plains 2026 Datasets/session_friction_scores.csv'

# Generate customer personas (required for Customer Simulations page)
python3 mirofish_export_pipeline.py --customers-only

# Generate all MiroFish export artifacts
python3 mirofish_export_pipeline.py
```

### Running the app

```bash
python3 -m streamlit run DataForce.py
```
